## 🚀 Overview

**Birthday Wisher** is a creative birthday greeting web app that blends **animations, audio, and interaction** to deliver a personalized digital celebration.

Instead of static wishes, users get a **fun, emotional, and engaging experience**.

## 🏗 How It Works

```text
Open page
   ↓
UI loads (cake + message)
   ↓
User interacts (click/blow)
   ↓
Animations trigger 🎉
   ↓
Audio + visuals create experience
```
